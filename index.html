Local Browser Tool

miku-docx2md

Version 0.8.1 / Updated 2026-04-24

What is this?

miku-docx2md は、Word の .docx ファイルをローカルで読み込み、文書構造を Markdown として抽出するツールです。

  • ブラウザ内で動作し、サーバ通信を行いません。
  • 段落、見出し、リスト、表、リンクを Markdown として取り出します。
  • 解決可能な埋め込み画像は sidecar asset として出力できます。

Use Cases

  • Word 文書を生成AIに渡しやすい Markdown に変換したい。
  • 仕様書、議事録、手順書などの文書構造を再利用したい。
  • 機密文書をアップロードせず、手元のブラウザだけで処理したい。

How to use

  1. Web ブラウザで miku-docx2md.html を開きます。
  2. .docx ファイルを選択します。
  3. 変換を実行し、Markdown と summary を確認します。
  4. Markdown、summary、画像 asset ZIP を必要に応じて保存します。

Features

Markdown conversion

見出し、段落、inline formatting、リンク、list、table を Markdown / HTML として出力します。

Asset export

解決可能な埋め込み画像を ZIP または CLI の sidecar directory として出力できます。

Diagnostics

summary と debug comment で、unsupported 要素や画像参照を確認できます。

Local processing

ブラウザ版はローカルで処理します。CLI も Node.js でローカル実行できます。
